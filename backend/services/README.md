# Service Layer (Business Logic Layer)
Used for data prosessing. Contains business rules and application logic.
## Responsibilities
- Ensuring data is correctly prosessed before database operations
- Executing calculations and data transformations
- Error handling
- Coordinating between [controllers](https://github.com/Osoito/ReadingDiplomaWebGame_Capstone1/tree/main/backend/controllers) and [models](https://github.com/Osoito/ReadingDiplomaWebGame_Capstone1/tree/main/backend/models)
