# Controller Layer (Presentation Layer)
Contains handlers and serves as the backend entry point. 
Handles incoming requests from the frontend and returns responses.
Sends request data to the [services](https://github.com/Osoito/ReadingDiplomaWebGame_Capstone1/tree/main/backend/services) for prosessing.
## Responsibilities
- Define additional endpoints
- Receive HTTP requests
- input validation
- convert data into suitable format for frontend
